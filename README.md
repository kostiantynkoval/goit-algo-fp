## Фінальний проєкт
### Завдання 7. Використання методу Монте-Карло

Отримані за допомогою методу Монте-Карло ймовірності майже співпадають з теоретичними значеннями. 
Це показує, що метод Монте-Карло працює правильно, якщо кількість симуляцій є достатньо великою.
Невеликі відхилення пояснюються випадковістю результатів. При збільшенні кількості кидків ці відхилення зменшуються. 
Отже, метод Монте-Карло є надійним способом оцінки ймовірностей і підтверджує правильність теоретичних розрахунків.
